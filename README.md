# DISEM - End-to-End Embedded AI Project

## Project Structure
- `Arduino/` - Embedded code for microcontrollers
- `MATLAB/` - Simulation and analysis scripts
- `Python/` - AI training and inference scripts
- `dnn3_evaluation/` - DNN model evaluation results
- `out_dnn3_fused/` - Optimized fused models
- `Rapport/` - Project documentation and report

## Datasets
The datasets (3500 CSV files, ~4GB) are available on Google Drive:
[(https://drive.google.com/drive/folders/1QOn2xHfqTe-rUGxV_5t3-_Vc-0Ma1N_i?usp=sharing)]

## Quick Start
1. Download datasets from the link above
2. Place in `dataset_csv_500x7/` folder
3. Run Python scripts for training
4. Use Arduino code for deployment

## Requirements
- Python 3.8+
- MATLAB R2020+
- Arduino IDE
